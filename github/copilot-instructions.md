При работе с HTML и CSS всегда используй методологию БЭМ (Блок, Элемент, Модификатор) для именования классов.
Используй два нижних подчеркивания для элементов (`block__element`) и два дефиса для модификаторов (`block__element--modifier`).
Никогда не делай глубокую вложенность вроде `block__elem1__elem2`. Структура классов всегда должна быть плоской.
Предлагай автодополнение тегов и классов в HTML строго по этой логике.
При написании или редактировании CSS-файлов ВСЕГДА проверяй классы на соответствие открытым HTML-файлам в проекте.
Если я пишу класс в CSS с опечаткой, путаю буквы или пишу имя, которого нет в HTML, твоя главная задача — мгновенно предложить автоисправление на точное и правильное имя класса из HTML. Не придумывай новые классы, если они не соответствуют разметке.
